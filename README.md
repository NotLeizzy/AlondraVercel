# Alondra's Bodega - Shoe Shop Website

A modern, responsive website for Alondra's Shoe Shop in Davao, showcasing premium footwear products.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern Typography** - Uses Sora and Outfit fonts for a contemporary look
- **Color Scheme** - Professional blue, red, and gold color palette
- **Product Showcase** - Grid layout for displaying shoe products
- **Call-to-Action** - Direct messaging link to Facebook
- **Footer Section** - Business information and quick links

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5.3.8
- Font Awesome Icons
- Google Fonts (Sora, Outfit)

## Deployment

This project is configured for Vercel deployment.

### To Deploy:

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Deploy on Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Click "Add New" → "Project"
   - Import your GitHub repository
   - Click "Deploy"

### Configuration

The `vercel.json` file handles:
- Static site configuration
- Route handling
- Clean URLs

## Customization

Update these files with your information:
- **Main.html** - Product details, contact info, Facebook link
- **style.css** - Colors, fonts, styling
- **Images** - Replace placeholder images with actual product photos
